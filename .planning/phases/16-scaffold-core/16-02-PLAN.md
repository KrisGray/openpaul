Replace the placeholder action handler (the "Phase 16 will implement" comment section) with full scaffolding implementation.

Imports to add:
```typescript
import { input, confirm } from '@inquirer/prompts'
import { existsSync } from 'fs'
import { resolve, join } from 'path'
import { getDefaultProjectName, createOpenPaulDir, generateStateJson } from './cli/scaffold.js'
import { info, step, success } from './cli/output.js'
```

Action handler logic (inside .action async callback):

1. Resolve target path:
   ```typescript
   const targetPath = resolve(options.path)
   const openpaulDir = join(targetPath, '.openpaul')
   ```

2. Check for existing directory (only if not --force):
   ```typescript
   if (existsSync(openpaulDir) && !options.force) {
     const shouldOverwrite = await confirm({
       message: '`.openpaul/` already exists. Overwrite?',
       default: false
     })
     if (!shouldOverwrite) {
       info('Operation cancelled')
       process.exit(0)  // Exit 0 - user cancelled, not error
     }
   }
   ```

3. Get project name (prompt only if --name not provided):
   ```typescript
   let projectName = options.name
   if (!projectName) {
     const defaultName = getDefaultProjectName(targetPath)
     projectName = await input({
       message: 'Project name',
       default: defaultName,
       validate: (value: string) => {
         if (!value.trim()) return 'Project name cannot be empty'
         if (/[\/\\:]/.test(value)) return 'Project name cannot contain /, \\, or :'
         return true
       }
     })
   }
   ```

4. Confirmation prompt (skip if --force):
   ```typescript
   if (!options.force) {
     const confirmed = await confirm({
       message: `Create OpenPAUL in '${targetPath}' with project name '${projectName}'?`,
       default: true
     })
     if (!confirmed) {
       info('Operation cancelled')
       process.exit(0)
     }
   }
   ```

5. Execute scaffolding:
   ```typescript
   createOpenPaulDir(targetPath)
   await generateStateJson(openpaulDir, projectName, pkg.version)
   success('OpenPAUL initialized successfully!')
   ```

Critical details:
- Use `process.exit(0)` for user cancellation (not exit(1) - cancellation is not an error)
- Validate project name with regex `/[\/\\:]/` to reject problematic characters
- Skip all prompts when `--force` is true
- The `info()` function only outputs when verbosity >= 1
