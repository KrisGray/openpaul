```typescript

}
```

Create ModelConfigManager class:

- Private cached: ModelConfigFile | null = null

Private load() method:
- Returns cached config if available
- Reads from fileManager.readModelConfig()
- Throws clear error if config not found: "Model configuration not found. Run /paul:init to initialize PAUL and create default configuration."
- Caches and returns the config

getModel(subStage: SubStage): ModelConfig method:
- Loads config via load()
- Returns config.subStages[subStage] ?? config.default (nullish coalescing)

async setModel(subStage: SubStage, modelConfig: ModelConfig): Promise<void>:

- Sets config.subStages[subStage] = modelConfig
- Updates lastUpdated timestamp
- Writes via fileManager.writeModelConfig()
- Updates cache

async setDefault(modelConfig: ModelConfig): Promise<void>:

clearCache(): void:
- Sets this.cached = null

async initializeDefault(): Promise<void>:
- Creates default config via createDefaultModelConfig()
- Ensures .paul directory exists via fileManager.ensurePaulDir()
- Writes config
- Caches result
