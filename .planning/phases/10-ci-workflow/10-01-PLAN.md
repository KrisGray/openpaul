Create GitHub Actions workflow at `.github/workflows/test.yml` with:

**Triggers (CI-01, CI-02):**
- `push` on branches: `main`, `develop`
- `push` on tags: `v*` (for release validation)
- `pull_request` on all branches with `types: [opened, synchronize, reopened]`
- Add conditional to skip draft PRs: `if: github.event.pull_request.draft == false` on job level

**Concurrency (CI-03):**
```yaml
concurrency:
  group: ci-${{ github.ref }}
  cancel-in-progress: true
```

**Matrix Strategy (OpenCode discretion):**
- Node.js: [20.x] (LTS, single version for simplicity)
- OS: [ubuntu-latest] (standard for JS projects)
- `fail-fast: false` per CONTEXT.md decision

**Job Steps:**
1. `actions/checkout@v4` with `fetch-depth: 1` (shallow clone)
2. `actions/setup-node@v4` with node-version from matrix, enable npm cache
3. `npm ci` (clean install)
4. `npm run test:coverage` (CI-04)
5. `actions/upload-artifact@v4` (CI-05):
   - name: `coverage-report`
   - path: `coverage/lcov.info`
   - retention-days: 7

Do NOT add:
- Path filtering (CONTEXT.md explicitly says no)
- Additional OS/Node versions (keep simple for now)
- Custom timeout (defaults are sufficient)
- Slack/email notifications (not in scope)
