   - Build Plan object:

     - plan: `${plan}` (zero-padded)
     - type: 'execute' (default)
     - wave: 1 (auto-detected from depends_on, default 1)
     - depends_on: [] (empty for now)
     - files_modified: flatten all task.files
     - autonomous: true (no checkpoints by default)
     - requirements: [] (empty for now)
     - tasks: from args
     - must_haves: { truths: [], artifacts: [], key_links: [] } (empty for now)
   - Call `fileManager.ensurePhasesDir()`
   - Call `fileManager.writePlan(phase, plan, planObject)`
   - Update state: `stateManager.savePhaseState(phase, { phase: 'PLAN', ... })`

   Output format (from CONTEXT.md):
   - Header: `# 📋 Plan: ${phase}-${plan}`
   - Meta: `**Type:** ${type} | **Wave:** ${wave} | **Tasks:** ${tasks.length}`
   - Default view: numbered task list with name + one-line description
   - Verbose view: full task details with files, verify, done
