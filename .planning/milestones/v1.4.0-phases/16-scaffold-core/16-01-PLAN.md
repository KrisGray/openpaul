Create src/cli/scaffold.ts with the following exported functions:

   - This handles edge cases like `./app` and `.` correctly

   - Create `.openpaul/` directory using `mkdirSync()` with `{ recursive: true }`
   - Import `existsSync`, `mkdirSync` from 'fs'
   - Return the created directory path
   - Use `step()` from './output.js' to log "Creating .openpaul/ directory..."

3. `generateStateJson(openpaulDir: string, projectName: string, cliVersion: string): Promise<void>`
   - Create StateFile object with:
     - version: "1.0"
     - cliVersion: passed parameter
     - name: passed parameter
     - createdAt: `new Date().toISOString()`
     - updatedAt: `new Date().toISOString()`
   - Use `atomicWrite()` from '../storage/atomic-writes.js' to write to `join(openpaulDir, 'state.json')`
   - Serialize with `JSON.stringify(state, null, 2)` for pretty printing
   - Use `step()` to log "Creating state.json..."

Imports needed:
- `resolve, basename, join` from 'path'
- `existsSync, mkdirSync` from 'fs'
- `atomicWrite` from '../storage/atomic-writes.js'
- `step` from './output.js'
- `StateFile` from '../types/state-file.js'

Do NOT create model-config.json - that's Phase 17.
Do NOT include interactive prompts in this module - prompts go in CLI action handler.
