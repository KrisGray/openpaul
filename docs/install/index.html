Installation Guide

Get OpenPAUL running in your project.

Quick Start

npx openpaul

This scaffolds the .openpaul/ directory with everything you need.

CLI Options

  • -n, --name - Project name (defaults to directory name)
  • -p, --path - Target directory (default: current)
  • --preset - Template preset: minimal or full
  • -f, --force - Skip prompts and overwrite existing files
  • -i, --interactive - Force interactive mode
  • -v, --verbose - Enable verbose output

Examples

npx openpaul                           # Interactive mode
npx openpaul --name my-project         # Skip name prompt
npx openpaul --path ./app              # Target directory
npx openpaul -n my-project -p ./app    # Combined options
npx openpaul --preset minimal          # Minimal template
npx openpaul --preset full             # Full template

What Gets Created

After npx openpaul:

.openpaul/
└── state.json       # Project registry (name, version, timestamps)
.opencode/
├── opencode.json    # OpenCode schema reference
└── (preset files)   # Commands, rules, agents — varies by preset

After /openpaul:init in OpenCode (initializes loop state and context):

.openpaul/
├── state.json           # Project registry
├── model-config.json    # Model configuration
├── state-phase-1.json   # Initial loop state
├── PROJECT.md           # Project context (from conversation)
├── ROADMAP.md           # Phase structure skeleton
├── STATE.md             # Loop position tracker
└── phases/              # Plans and summaries (created as needed)

Install as OpenCode Plugin

To use OpenPAUL commands inside OpenCode (like /openpaul:plan):

  1. Open or create ~/.config/opencode/opencode.json:
{
  "plugins": {
    "npm": [
      "openpaul"
    ]
  }
}
  1. Restart OpenCode to load the plugin.

Verify Installation

  • CLI: Run npx openpaul --version
  • Plugin: Run /openpaul:help inside OpenCode

With OpenCARL (Recommended)

For the best experience, use OpenPAUL with OpenCARL:

# Initialize OpenPAUL
npx openpaul

# Initialize OpenCARL
npx opencarl --local   # Current project only
npx opencarl --global  # All projects

OpenCARL auto-loads OpenPAUL rules when it detects .openpaul/ directory.
